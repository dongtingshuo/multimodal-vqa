# v0.2.0 - Staged Cross-Attention Checkpoint

This release publishes the completed 12-epoch staged cross-attention model as the recommended stable checkpoint for `multimodal-vqa`.

本版本发布完成 12 epoch 训练的 staged cross-attention 模型，作为 `multimodal-vqa` 当前推荐的稳定 checkpoint。

## Validation Metrics / 验证指标

| Metric | Value |
| --- | ---: |
| Best epoch | 12 |
| Validation loss | 4.328347 |
| Hard Top-1 accuracy | 0.523929 |
| Internal VQA score | 0.623305 |
| Top-5 VQA score | 0.863991 |
| Evaluated validation examples | 209,410 |
| Exported validation predictions | 214,124 |

These values are produced by the project evaluator on the documented VQA v2 / COCO 2014 workflow. They are not official VQA leaderboard scores.

以上结果由项目 evaluator 在文档规定的 VQA v2 / COCO 2014 流程上生成，并非官方 VQA leaderboard 分数。

## Scope / 使用范围

The checkpoint supports inference, evaluation, the Gradio demo, and compatible continuation training. PyTorch checkpoints use pickle-based serialization; download only from the official Release and verify the checksum before loading.

该权重支持推理、评估、Gradio 演示和兼容配置下的继续训练。PyTorch checkpoint 使用基于 pickle 的序列化格式；请仅从官方 Release 下载，并在加载前校验哈希。

The ViLT seed-42 result is not included in this release because its configured training run and official evaluation are incomplete.

本版本不包含 ViLT seed 42 权重，因为其配置的训练任务与官方评估尚未完成。

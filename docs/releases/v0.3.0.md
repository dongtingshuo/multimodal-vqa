# v0.3.0 - ViLT Full-Validation Checkpoint

This release publishes the completed seed-42 ViLT model as the recommended engineering checkpoint for `multimodal-vqa`.

本版本发布完成 seed 42 训练的 ViLT 模型，作为 `multimodal-vqa` 的推荐工程 checkpoint。

## Training Protocol / 训练协议

- Seed: 42
- Data: complete VQA v2 train/val questions and annotations with COCO 2014 images
- Train/validation examples: 434,635 / 209,608 internally scorable examples
- Effective batch size: 32 (`batch_size=4`, accumulation 8)
- Selection metric: internal VQA score
- Schedule: warmup + plateau reduction, early-stopping patience 2
- Hardware: Kaggle P100 for epochs 1-2; AutoDL RTX 4090D for continuation through epoch 7
- Total recorded training time: 96,989 seconds

## Scope / 使用范围

The checkpoint supports inference, internal evaluation, the Gradio demo, and compatible continuation training. Runtime construction also requires the ViLT Hugging Face backbone and processor assets. PyTorch checkpoints use pickle-based serialization; download only from the official Release and verify the checksum before loading.

该权重支持推理、内部评估、Gradio 演示和兼容续训。运行时还需要对应的 Hugging Face ViLT backbone 与 processor 资源。PyTorch checkpoint 使用基于 pickle 的序列化格式；请仅从官方 Release 下载，并在加载前校验哈希。

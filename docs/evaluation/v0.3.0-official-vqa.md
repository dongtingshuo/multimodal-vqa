## Compatibility / 兼容性

The official repository is written for Python 2. The project preparation script applies the
standard `2to3` syntax conversion and repairs the repository's known mixed-indentation lines in a
working copy. No answer normalization, matching, aggregation, or rounding logic is changed.

官方仓库面向 Python 2。项目准备脚本会在工作副本中执行标准 `2to3` 语法转换，并修复仓库中
已知的混合缩进行；答案规范化、匹配、聚合和舍入逻辑均未修改。

```bash
git clone --depth 1 https://github.com/GT-Vision-Lab/VQA.git /tmp/VQA
python scripts/prepare_official_vqa_toolkit.py --toolkit-root /tmp/VQA
python scripts/run_official_vqa_eval.py \
  --toolkit-root /tmp/VQA \
  --questions data/vqa/v2_OpenEnded_mscoco_val2014_questions.json \
  --annotations data/vqa/v2_mscoco_val2014_annotations.json \
  --predictions outputs/autodl-vilt-seed42/val_predictions.json \
  --output outputs/autodl-vilt-seed42/official_vqa_metrics.json
```

## Metric Difference / 指标差异

The project-internal VQA score is `71.01`, while the official validation score is `68.42`. The
internal metric covers 209,608 examples scorable under the Top-3000 vocabulary; the official
toolkit scores all 214,354 validation predictions with its canonical answer normalization and
leave-one-annotator-out agreement calculation. The official value should be used for external
protocol comparisons.

项目内部 VQA score 为 `71.01`，官方验证得分为 `68.42`。内部指标只覆盖 Top-3000 词表下可评分
的 209,608 条样本；官方 toolkit 使用规范答案处理和逐标注者留一一致性计算，对全部 214,354 条
预测评分。进行外部协议对比时应使用官方值。
